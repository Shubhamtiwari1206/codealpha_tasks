# 🧮 CodeAlpha Calculator

A simple and responsive calculator built using HTML, CSS, and JavaScript as part of the CodeAlpha Frontend Development Internship.

## 🚀 Features

- Addition (+)
- Subtraction (-)
- Multiplication (×)
- Division (÷)
- Clear Screen Function
- Delete Last Character
- Responsive Design
- Keyboard Support
- Modern User Interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

```
CodeAlpha_Calculator/
│
├── index.html
└── README.md
```

## 🎯 How to Use

1. Open the project in a web browser.
2. Click the buttons to perform calculations.
3. Use your keyboard for quick input.
4. Press:
   - Enter → Calculate Result
   - Backspace → Delete Last Character
   - Esc → Clear Screen
