# 🖼️ CodeAlpha Image Gallery

A responsive Image Gallery built using HTML, CSS, and JavaScript as part of the CodeAlpha Frontend Development Internship.

## 🚀 Features

- Responsive Image Gallery Layout
- Next & Previous Navigation Buttons
- Lightbox Image Preview
- Smooth Hover Effects
- CSS Animations & Transitions
- Mobile-Friendly Design
- Category Filtering (Bonus)
- Modern User Interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

```
CodeAlpha_ImageGallery/
│
├── index.html
└── README.md
```

## 🎯 How to Use

1. Open the project in a web browser.
2. Browse the gallery images.
3. Click an image to open it in Lightbox View.
4. Use Previous and Next buttons to navigate.
5. Use category filters (if available) to sort images.
